# EV Battery Pack Simulation (79 kWh LMFP)

## Overview
This project models a high-voltage EV battery pack using LMFP chemistry in MATLAB Simulink.

## Specifications
- 108 cells in series
- 235 Ah capacity
- ~400 V nominal voltage
- ~79–80 kWh usable energy

## Features
- Battery discharge simulation (200 A load)
- Voltage monitoring and analysis
- BMS cutoff logic (270–280 V hysteresis)
- Realistic LMFP discharge behavior

## Tools Used
- MATLAB
- Simulink
- Simscape Electrical

## Results
The simulation shows:
- Initial voltage drop due to internal resistance
- Plateau region during discharge
- Cutoff at ~270 V
